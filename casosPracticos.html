HiddenEye

Actualizado: Abril 14, 2020

¿Qué es HiddenEye?

Con Hidden Eye podemos realizar phishing de las páginas más conocidas. También existe la posibilidad de introducir las plantillas de webs diferentes a las ofrecidas por la herramienta para poder suplantarlas también.

Es decir, podemos suplantar cualquier web siempre y cuando le proporcionemos al programa las platillas necesarias. A parte de suplantar las páginas web, existe la posibilidad de incluir un keylogger que registre las pulsaciones introducidas por el usuario en la propia web suplantada.

Ngrok

Actualizado: Abril 14, 2020

¿Qué es Ngrok?

Ngrok nos permite exponer a internet una URL generada dinámicamente, la cual apunta a un servicio web que se está ejecutando en nuestra máquina local. Por ejemplo: si tenemos un servicio web arrancado en http://localhost:8080, ngrok genera dinámicamente una URL del tipo http://xxxxxx.ngrok.io visible en internet, y que apunta directamente a nuestro localhost.

¿Qué es NMAP?

Nmap se utiliza para descubrir hosts y servicios en una red informática enviando paquetes y analizando las respuestas.

Nmap proporciona una serie de funciones para sondear redes informáticas, incluido el descubrimiento de hosts y la detección de puertos, servicios y sistemas operativos . Estas funciones se pueden ampliar mediante scripts que proporcionan una detección de servicios más avanzada, detección de vulnerabilidades, y otras funciones.

NMAP ya viene instalado por defecto en Kali Linux.

Pre-requisitos

Aberrirguar nuestra IP con el comando:
$ if config



Con la IP y NMAP podemos escanear nuestros puertos y puertos de dispositivos en nuestra red.

Detecion de dispositivos en mi red

Ejecutamos en la terminal el siguiente comando, dando un rango de IP grande para así detectar los dispositivos que están conectados a ella:
$ sudo nmap -sP TU_IP-254

Escaneo de puertos

Ejecutamos en la terminal el siguiente comando, para ver los puertos abiertos que puedrian ser vulnerabilidades:
$ sudo nmap TU_IP/24
/24: Indica el escaneo de todos los puertos disponibles
